# 🧘 Yoga Pose Explorer

A simple JavaScript web application for exploring yoga poses using the **Yoga API**. Search through poses, browse them with pagination, and view detailed information about each pose.

## ✨ Features

- 🔍 Search yoga poses by name
- 📄 Pagination for easy browsing
- 🖼️ Pose images
- 📖 Detailed page for each pose
- 📱 Responsive interface built with Bootstrap
- ⚡ Built with vanilla JavaScript (no frameworks)

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Fetch API

## 📡 API

This project uses the open-source **Yoga API**:

https://github.com/alexcumplido/yoga-api
